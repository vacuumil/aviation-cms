# Aviation Safety CMS

CMS для сайта по безопасности в авиации. Использует Strapi + PostgreSQL.

## Быстрый старт

1. Установите зависимости: `npm install`
2. Запустите в development режиме: `npm run develop`
3. Откройте http://localhost:1337/admin

## Деплой на Railway

Проект автоматически деплоится при пуше в main ветку.
